World Wide Web

En informatica, la World Wide (WWW) o Red informatica mundial comunmente conocida como la web, es un sistema de distribucion de documentos de hipertex o hipermedios interconectados y accesibles via internet.con un navegador web, un usuario visualiza sitios web compuestos de paginas web que pueden contener texto, imagenes, videos u otros contenidos multimedia, y navega a traves de esas paginas usando hiperenlaces.

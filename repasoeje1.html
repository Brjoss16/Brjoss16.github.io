Los tres pilares de la web

Tim Tim_Berners-Lee es considerado el padre de las paginas web porque desarrollo los tres elementos basicos para el funcionamiento de la Web

  • HTML
  • HTTP
  • URL

    • HTML

    HTML, siglas de HyperText Markup Language, hace referencia al lenguaje de marcado para la elaboracion de paginas web. Es un estandar que sirve de referencia para la elaboracionde paginas web en sus diferentes versiones, define una estructura basica y un codigo (denomido código HTML) para la definicion de contenido de una pagina web, como texto, imagenes, entre otros. Es un estandar a cargo de la W3C, organizacion dedicada a la estandarizacion de casi todas las tecnologias ligadas a la web, sobre todo lo que hace referencia a su escritura e interpretacion. Es es lenguaje con el que se define las paginas web.

    HTTP

    HyperText Transfer Protocol o HTTP(en español protocolo de transferencia de hipertexto) es el protocolo usado en cada transaccion de la Word Wide Web fue desarrollado por el Worl Wid Web Consortium y la Internet Engienering Task Force, colaboracion que culmino en 1999 con la publicacion de una serie de RFC, el más importante de ellos es el RFC 2616 que especifica la version 1.1

    URL

    Un Localizador de recursos uniforme o URL -siglas en ingles de Uniform Resource Locator- es un identificador de recursos uniformes (URL) cuyos recursos referidos pueden cambiar, esto es, la direccion puede apuntar a recursos variables en el tiempo. Estan formados por una secuencia de carcateres, de acuerdo a un formato modelico y estandar, que designa recursos en una red, como Internet.
